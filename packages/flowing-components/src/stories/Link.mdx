## 注意事项

1. 当设置 `href` 时，组件渲染为 `<a>` 标签，否则渲染为 `<span>` 标签
2. 当设置 `icon={true}` 时，将显示默认的链接图标
3. 禁用状态下，链接将不可点击且样式变为半透明
4. 所有链接默认在新窗口打开（target="\_blank"）
