# Divider API

区分内容的分割线组件。

## 何时使用

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割，例如表格的操作列。

## 注意事项

1. 水平分割线可以包含文字，垂直分割线不支持文字
2. 当包含文字时，分割线会自动调整为 flex 布局
3. 垂直分割线高度默认为当前行高的 0.9em
4. 分割线默认使用 slate-200 颜色，可通过 style 自定义
