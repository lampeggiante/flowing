## 图标尺寸

通过 Tailwind 的文本尺寸类来控制图标大小：

- `text-xs`: 12px
- `text-sm`: 14px
- `text-base`: 16px
- `text-lg`: 18px
- `text-xl`: 20px
- `text-2xl`: 24px

## 图标颜色

通过 Tailwind 的文本颜色类来控制图标颜色：

- `text-fuchsia-600`: 主色
- `text-emerald-600`: 成功
- `text-yellow-600`: 警告
- `text-red-600`: 错误
- `text-slate-600`: 默认

## 动画效果

使用 Tailwind 的 `animate-spin` 类来添加旋转动画。
